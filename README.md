A collection of my favorite cat ascii art.

## ASCII -> PNG convertor
### installation
```
pip install -r requirements.txt
```

### Usage
`{ascii_id}` can be set to `*` to convert all in `./ascii/` to images.
```
python3 ./convert_a2i.py {ascii_id} {*font_size}
```
